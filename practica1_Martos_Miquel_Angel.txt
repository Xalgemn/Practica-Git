comandos utilizados:

git branch (Comprobar las ramas)
git branch -a (Comprobar TODAS las ramas, incluyendo las que no se ven con el comando anterior)
git checkout (cambiar de rama)
git status (comprobar el estado de los archivos)
git add (preparar archivos para el commit)
git commit -m "Nombre" (Enviar archivos al repositorio local)
git push --set-upstream origin "Directory" (Subes un push de la rama de tal forma que desde gitHub puedes tener ambas ramas, tanto la principal como la que has subido con cambios)
cd (cambiar de directorio)
cd .. (volver al directorio anterior)
git merge (juntar dos ramas)
git remote add origin https://github.com/User/Directory.git (Crea un "origin" en caso de que este no exista en el repositorio remoto)
git remote rename origin upstream (Cambia el nombre de un repositorio)
